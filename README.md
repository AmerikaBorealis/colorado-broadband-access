# colorado-broadband-access

A spatial data pipeline joining FCC Broadband Data Collection (47K records) with Census TIGER place geometries for all 482 Colorado municipalities, producing an enriched GeoJSON visualizing residential broadband coverage at 6 speed tiers.

Deployed as an interactive choropleth map with graduated symbology on 25/3 Mbps and 100/20 Mbps FCC access thresholds; available at https://alexraugeo.github.io/colorado-broadband-access/
